üìÑInput File Specifications

The system requires three CSV files with the following exact formats:

1. Teaching Assistant Availability (Maximum Hours)

TA Name, Hired For
Example:
John Smith, 20
Jane Doe, 15
Mike Johnson, 25

2. Availability Responses (Time Preferences)

Name, [Unavailability columns...]
Example:
Name, Unavailable[Tuesday 8-11], Unavailable[Tuesday 11-14], ...
John Smith, No, Yes, No, ...
Jane Doe, Yes, No, No, ...

3. Laboratory Requirements (Schedule Definition)

Lab Section, Day, Start, End, Required
Example:
LAB 801, Tuesday, 14, 17, 4
LAB 802, Thursday, 14, 17, 3
LAB D01, Tuesday, 8, 11, 2
Critical: Column headers must match exactly as specified. Time values must be in 24-hour format (e.g., 14 = 2:00 PM).
