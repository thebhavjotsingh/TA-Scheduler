Getting Started

  1. Install Dependencies: Run pip install -r requirements-minimal.txt for essential packages only
  2. Launch Application: Execute python main.py from the project directory
  3. Prepare CSV Files: Ensure your three required CSV files match the formats specified below
  4. Upload Files: Use the modern drag-and-drop interface or browse buttons
  5. Configure Constraints: Set daily hour limits, min/max labs per TA (defaults: 4h/day, 1-3 labs range)
  6. Run Optimization: Choose "Run Scheduler" for best single solution, or "Find Alternative Solutions" for multiple options
  7. Review Results: Analyze slot coverage, TA assignments, and utilization statistics
  8. Export Data: Export CSV summaries and generate interactive Excel schedules with VBA automation
New in v3.0: Interactive Excel schedule generation with real-time Office Hours tracking and VBA automation support.

Input File Specifications

The system requires three CSV files with the following exact formats:

1. Max Availability CSV (max availability.csv)

TA Name,Hired For
Purpose: Defines TA names and maximum weekly hours
Example:
Devipriya Raju,20
Moriom Chowdhury Kumu,15
Sai Vashnavi Jattu,25

2. Responses CSV (responses.csv)

Timestamp,Email Address,Name, [8am to 9am], [9am to 10am], [10am to 11am], ...
Purpose: TA availability responses with unavailable time slots
Example:
7/28/2025 15:48:18,email@ualberta.ca,John Smith,"Monday, Tuesday","","Wednesday, Friday"
Note: Unavailable days are listed in each time slot column (comma-separated)

3. Requirements CSV (requirements.csv)

Lab Section,Day,Start,End,Required
Purpose: Lab section definitions with staffing requirements
Example:
LAB 801,Tuesday,14,17,4
LAB 802,Thursday,14,17,3
LAB D01,Tuesday,8,11,2
Critical Requirements:
‚Ä¢ Column headers must match exactly as specified
‚Ä¢ Time values must be in 24-hour format (e.g., 14 = 2:00 PM)
‚Ä¢ Responses CSV: unavailable days listed as comma-separated values
‚Ä¢ File names can be different but formats must be exact

Technical Support & Troubleshooting

Common Issues and Solutions:

  • Installation Issues: Use requirements-minimal.txt for essential packages only, or check INSTALLATION.md for detailed setup
  • File Format Errors: Verify CSV column headers match specifications exactly (case-sensitive)
  • No Available Staff: Review availability data for scheduling conflicts or missing responses
  • Constraint Issues: Consider adjusting daily hour limits or max labs per TA parameters
  • Excel Generation Problems: Ensure openpyxl is installed and output directory has write permissions
  • Missing TA Data: Check the system log for unreported availability submissions
